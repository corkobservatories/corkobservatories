The *.par and *.eerom files contain the parameter settings of installed MT-01/RTC/PPC measurement systems.

The files are named as follows:

InstrumentType_Site_InstallationDate.[par|eerom]

Examples:
BPR_NEPTUNE-889_200909.par

== InstrumentType
Either CORK or BPR

== Site
E.g  IODP-U1364, ODP-857, NEPTUNE-Barkley, ...

== InstallationDate
Year and month should be sufficient 200909
