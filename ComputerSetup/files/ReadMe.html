How to get help

You can find (and provide) help on the CORK website http://www.corkobservatories.org. A local version of the wiki is installed on this computer: http://localhost/www/pmwiki/index.php

NOTE: This is only for reference, when no internet connection is available. All changes you make to this wiki will be lost after the next syncronization!

This is how you update the local wiki:

sudo rsync -ave ssh --delete --exclude=www/tmp corks@mercury.uvic.ca:~/www /srv/www/htdocs

enter the passwd twice..

Before you start...

Before you use mlterm or play around, please make your self a directory to do this and cd into that directory. The command is mkdir. If you do testing create a subdirectory under Tests (open terminal; “cd Tests”; “mkdir myNewDir”; “cd myNewDir”; start playing). If you are on a cruise create a subdirectory under Cruises...

You can find the necessary parameter files under the “corkobservatory” directory. Copy the files you need from there into your current working dir (“cp ~/corkobservatory/CalibrationFiles/*.txt .” if you are in your working dir).

To update the parameter files, etc with the current version on the internet (“cd corkobservatory”,”svn update”). Type “svn commit” to push changes you made under the corkobservatory directory to the internet...
